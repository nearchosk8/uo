# Untitled Office — website

Astro + Tailwind v4 portfolio site for Untitled Office (Athens, GR).

## Run locally
```bash
npm install        # first time only
npm run dev        # → http://localhost:4321  (live, hot-reloads)
```

## Build for deploy
```bash
npm run build      # static output in dist/
npm run preview    # preview the production build locally
```

Deploys as a static site to Netlify, Vercel, or Cloudflare Pages
(point them at this repo; build command `npm run build`, output dir `dist`).

See `CLAUDE.md` for design intent, the theme system, fonts, and the list of
assets still to be supplied.
